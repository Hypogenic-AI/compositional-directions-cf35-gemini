# Cloned Repositories

## 1. park_geometry
- **URL**: https://github.com/KihoPark/linear_rep_geometry
- **Purpose**: Official code for "The Linear Representation Hypothesis and the Geometry of LLMs" (2023).
- **Key Files**: `linear_rep_geometry.py`, `word_pairs/`, `1_subspace.ipynb`.
- **Notes**: Use for base vector extraction and steering.

## 2. truthfulness_probes
- **URL**: https://github.com/colored-dye/truthfulness_probe_generalization
- **Purpose**: Code for "Probing the Geometry of Truth: Consistency and Generalization" (2025).
- **Key Files**: `code/train.py`, `data/` (contains logical transformation datasets).
- **Notes**: Essential for testing composition with logical operators.

## 3. tuned_lens
- **URL**: https://github.com/AlignmentResearch/tuned-lens
- **Purpose**: Library for eliciting latent predictions from hidden states.
- **Key Files**: `tuned_lens/nn/lenses.py`, `notebooks/prediction_trajectories.ipynb`.
- **Notes**: Use to analyze how compositionality develops through layers.

## 4. park_categorical
- **URL**: https://github.com/KihoPark/LLM_Categorical_Hierarchical_Representations
- **Purpose**: Official code for "The Geometry of Categorical and Hierarchical Concepts" (2024).
- **Key Files**: `data/`, WordNet based concept extraction.
- **Notes**: Useful for hierarchical composition experiments.

## 5. mlsae
- **URL**: https://github.com/tim-lawson/mlsae
- **Purpose**: Multi-layer Sparse Autoencoders.
- **Key Files**: `train_mlsae.py`.
- **Notes**: For analyzing cross-layer feature flow.
